# AI Assistant Chatbot

A simple command-line AI chatbot built using Python. This chatbot can respond to greetings, provide the current date and time, answer basic questions about AI, Python, and Machine Learning, tell jokes, perform calculations, and more.

## Features

- Greeting responses (hi, hello, hey)
- Introduces itself
- Shows current time
- Shows current date
- Provides information about Python
- Provides information about Artificial Intelligence
- Provides information about Machine Learning
- Motivational responses
- Tells programming jokes
- Performs basic calculations
- Exit functionality (bye, exit, quit)
- Handles unknown inputs gracefully

## Technologies Used

- Python 3
- datetime module

## Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/AI-Assistant-Chatbot.git
```

### Move to the Project Directory

```bash
cd AI-Assistant-Chatbot
```

### Run the Program

```bash
python chatbot.py
```

## Supported Commands

| Command | Description |
|-----------|-------------|
| hi, hello, hey | Greeting message |
| your name | Introduces the chatbot |
| time | Shows current time |
| date | Shows current date |
| python | Information about Python |
| ai | Information about Artificial Intelligence |
| machine learning | Information about Machine Learning |
| motivate me | Motivational quote |
| joke | Programming joke |
| calculate | Performs basic calculations |
| bye, exit, quit | Ends the conversation |
